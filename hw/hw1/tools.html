VirtualBox is a free tool that can be used to take advantage of virtualization and virtual machines. VirtualBox is owned by Oracle and is available for x86 hardware.

VMware is another company that provides virtualization tools and web services. VMware Workstation is an application that can be bought in order to create virtual machines.
