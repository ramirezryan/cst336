If you are interested in creating virtual machines, VirtualBox can be downloaded for free from Oracle and is compatible with Windows, OS X, Linux, and Solaris

New virtual machines can be created by clicking on the 'New' button in the top left corner of the VirtualBox window. The machine can be named, and the planned operating system selected.

After naming and selecting the operating system for the machine, resources must be allocated for the machine, starting with memory. The virtual machine's resource usage cannot exceed the amount of hardware resources available from the host machine.

A virtual hard disk will need to be used for the virtual machine. If you are creating a completely new virtual machine, you will probably want to create a new virtual hard disk rather than use an existing one.

Just like there are different formats for certain files, there are different formats for virtual disks. If you plan on using another virtualization tool (such as VMware Workstation), then you want to be conscious of the format picked here. If you only plan on using VirtualBox, then the standard VDI format is fine.

Here you have the option between a dynamically allocated virtual hard disk and a fixed size for the disk. Dynamically allocated disks will only use storage space when they absolutely need it, while fixed size disks reserve space on the disk for future use, even if that space is not currently being used.

Here is where you have the option to choose where the virtual disk file gets stored on the host machine. Additionally, here is where you can assign how much storage you would like for your virtual machine to have.

The virtual machine has finally been created! Only the operating system install remains.

Just like building a new PC, a boot medium is required so that the virtual machine knows what it needs to be running. Here is where you can provide a disk image (.iso) to be used as the boot medium. This is the final step in creating a virtual machine before getting into OS specific setup. Congratulations!
