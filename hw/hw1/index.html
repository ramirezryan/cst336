Welcome.

This website exists with the intention of explaining the concept and application of virtualization in computer science.

Virtualization is the process of a computer turning complicated hardware resources into virtual resources that can then be allocated for specific uses. Virtual machines take advantage of virtualization by further partitioning resources in order to create independent workspaces, allowing multiple operating systems and services to be run on each virtual machine.
